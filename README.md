## Hiring Process Analytics

This project focuses on analyzing hiring workflows to identify
friction points, inefficiencies, and opportunities for optimization.

The goal is to measure how process design impacts:
- Time to hire
- Candidate drop-off
- Recruiter workload
- Decision-making efficiency

This repository serves as a foundation for future analysis
comparing traditional hiring processes with AI-powered workflows.

### Focus areas
- Hiring funnel analysis
- Process efficiency metrics
- Interview workflow optimization

### Tools
- Data analysis
- Process metrics
- Power BI (planned)
